# Frontend

This directory contains the frontend code for the Attendance System.

## Setup

1. Install Node.js.
2. Run `npm install` to install dependencies.
3. Run `npm start` to start the development server.

## Configuration

- **API URL**: By default, `npm start` proxies `/api` requests to `http://localhost:5000`.
- **Netlify**: The `netlify.toml` file is configured for deployment. Update `REPLACE_WITH_YOUR_RENDER_BACKEND_URL` with your actual backend URL.

## Structure

- `index.html`: Main entry point.
- `assets/`: Static assets (CSS, JS).
- `Loginv2/`: Login page files.
- `Admin Page/`: Admin dashboard files.
- `Student page/`: Student dashboard files.
